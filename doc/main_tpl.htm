Шаблон main.tpl.

    

main.tpl

 

Данный шаблон является контейнером для контента галереи. По сути шаблон является аналогом main.tpl стандартного шаблона DLE, предполагается что взаимодействие шаблонов галереи и DLE  будет ограничено тегами [aviable], тоесть

1.{speedbar} – Контекстная навигация
2.{search_keyword} -   слово, по которому проводится поиск.
3.{alfa} - html разметка алфавита.
4.{content} - скомпилированный контент.
5.{pagination} - постраничная навигация.
6.{js_min} – динамическая ссылка на сжатые файлы javaScript.
7.{cat_tree} – html разметка, дерево категорий.
