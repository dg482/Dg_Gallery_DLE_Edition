Дополнительно.

    

Несколько советов по работе в админпанели компонента.

1.Не переключайте вкладки активного окна пир загрузке файлов.
2.Если загрузка не начинается (зависла) обновите окно браузера.
3.Если в ответ на ваше действие ничего не происходит, обновите окно браузера.
4.Обновляйте окно браузера после добавления ссылок с сервисы YouTube.

Если ошибка повторяется, обратитесь за помощью к разработчику, предоставив следующую информацию:

1.При каких действиях произошла ошибка.
2.В каком браузере произошла ошибка.
3.В какой Операционной Системе произошла ошибка.

По возможности прикрепите текс ошибки выводимый в консоли расширением Firebug для Firfox, либо сообщение отладчика  предусмотренного в вашем браузере.
