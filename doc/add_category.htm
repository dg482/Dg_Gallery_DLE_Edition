Категории.

    

Категории галереи.

 

Категории являются важной состовляющей в обоих предлогаемых конфигурациях скрипта. В конфигурации «Портал» в категориях размещаются альбомы пользователей, а в конфигурации «Архив» категории служат местом загрузки целвых файлов. У каждой категории есть собственные мета теги и параметры доступа. Категории могут быть не ограниченной вложенности, привилегии родительских категорий ни как не влияют на дочерние.

 

 

 

 

Для создания категории перейдите в раздел «новая категория» из списка разделов в левом меню админпанели скрипта.  Заполните поля и нажмите кнопку «отправить».  При создание категории обязательным полем является «Имя», все остальные поля не обязательны и заполняются по желанию. Обратите внимание на разделы  «Загрузка разрешена» и «Просмотр разрешен», в данных полях определяются   группы пользователей, которым разрешено либо запрещено указанное действие.

 

 

При щелчке правой кнопкой мыши по текстовому полю, доступны сервисные методы авто генерации ключевых слов и описания страницы.   

 

 

 

 

Для редактирования категории щелкните правой кнопкой мыши на название в дереве категорий и выберите необходимое действие.
