Общая информация.

    

 

В скрипте используется стандартный шаблонизатор DLE.  В каждой конфигурации используется сой шаблон профиля пользователей. В конфигурации «Портал» используется шаблон user_mode1.tpl и albom_edit.tpl,  в конфигурации «Альбом» только user_mode2.tpl.

 

В каждои конфигурации используется ограниченное кол-во тегов, однако в шаблоне Default/gallery/ используются практически все теги, это сделано для демонстрации возможностей скрипта на одном шаблоне.

 

Для снижения нагрузки рекомендуется удалить все проверки конфигураций, и не нужные в ваших шаблонах теги из методов представления php (app/view/).

 

 

Во всех шаблонах действуют глобальные теги DLE:

1.{include file="my_block.tpl"}
2.[group=1,2,3] текст [/group]
3.[not-group=1,2,3] текст [/not-group]
4.[aviable -gallery =…] текст [/aviable -gallery] – тег идеентичен [aviable=]
5.

В качестве параметра для тега aviable -gallery предусмотренны следующие значения:

1.аlbom – прои просмотре альбома.
2.category – при просмотре категории.
3.file – при просмотре отдельного файла.
4.search – при поиске по галерее.
5.profile – при просмотре профиля.
6.full – при просмотре отдельного файла.
7.show_file – при просмотре файлов в категоии.

 

При использование {include file="my_block.tpl"} файл my_block.tpl должен находится в директории шаблона скрипта ../templates/my_template/gallery/my_block.tpl
