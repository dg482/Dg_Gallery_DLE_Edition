Технические требования.

    

Минимальные технические требования к серверу.

1.Apache 2.0+*
2.PHP 5.2+*
3.MySQL 4.0+ или MySQL 5.0+.
4.Установленная библиотека PHP GD2.*
5.Установленная библиотека PHP iconv.*
6.Установленное расширение PHP FFMPEG**
7.Установленный ionCube loader 4+.*

 

Настройки PHP.

·Safe Mode: OFF*.
·File Uploads: ON*.
·Max Execution Time: 30***.
·Upload Max Filesize: 8M***.
·Memory Limit: 32M ***

 

 

 

*обязательные параметры.

**рекомендуемые параметры.
