Шаблон edit_albom.tpl.

    

Шаблон редактирования альбомов albom_edit.tpl

 

В данном шаблоне определяется внешний вид интерфейса редактирования альбома, загрузки файлов. Для правильной работы не изменяйте идентификаторы элементов разметки, имена элементов форм.  Данный шаблон дополняется HTML разметкой сгенерированной php (view_editAlbom::render())

 

1.{files} – HTML список файлов, настраивается в классе представления view_editAlbom::render();
2.[youtube][/youtube] – выводится если разрешено добавление ссылок с сервиса YouTube.
3.{form-access} – HTML субформа настроек доступа к просмотру альбома.
4.{form-edit} – HTML субформа редактироване описания альбома, ключевых слов.
