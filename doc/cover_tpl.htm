Шаблон cover.tpl.

    

 

cover.tpl

 

Шаблон предназначен для вывода превью файла или обложки альбома, категории.

1.[access_granted] [/access_granted] - доступ к материалу разрешен.
2.[access_denied] [/access_denied]  - доступ к материалу запрещен.
3.[category] [/category] – текст выводится при просмотре категории.
4.[albom] [/albom] - текст выводится при просмотре альбомов.
5.[file] [/file] - текст выводится при просмотре файлов.
6.[link] [/link] – ссылка на альбом, категорию в зависимости от контекста применения.
7.[catlink] [/catlink] – ссылка на категорию при просмотре альбомов.
8.{src} – ссылка на превью.
9.{name} – название категории, альбома.
10.{albom_num} – кол-во альбомов, только пр просмотре категорий.
11.{file_num} – кол-во файлов, только при просмотре альбомов.
12.{category} – категория, только при просмотре файла.
13.{width}x{height} - размеры, только при просмотре файла.
14.{date} – дата добавления, только при просмотре файла.

Разметка шаблона Default/gallery/cover.tpl

 

<div class="g-block"> [access_granted]

  [link]<img src="{src}" alt="{meta_name}   {meta_keyword}"/>[/link]

  [/access_granted]

  [access_denied]

  <div class="error b-4">

    <p>Доступ запрещен.</p>

  </div>

  [/access_denied]

  <div style="padding:5px;"> [category] <span class="grey">Категория:</span> [link]{name}[/link]<br />

    [/albom]

    [file]

    </div>
